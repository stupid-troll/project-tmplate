# Index template

```markdown
# <Section name> (Required)

**Type: <type>** (Required)

Short description (1-3 sentences). (Optional but recommended)

## Contents (Required — at least one entry)

- [Topic 1](./topic-1.md) — one-line description. (Recommended)
- [Topic 2](./topic-2.md) — one-line description. (Optional)
- [Subsection: Topic A](./subsection/index.md) — short note. (Optional)

## Common (Optional)

- [Commits](../common/delivery/commits.md) — Shared commit message standard.
```

Notes:

- Use relative links `./` and `../` for navigation inside the section and to common pages.
- Keep `Contents` descriptions short to make the section scannable.
